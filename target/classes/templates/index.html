ex-wallet-config

本地中心化交易所钱包的配置中心，当前聚焦币种与链扩展参数配置。

核心配置模块

币种配置

Core
维护币种基础信息：币种 ID、简写、全称、精度、图标、启用状态。
进入 /coin-config

区块链配置

Core
维护区块链简称与全称映射，用于币种扩展参数自动带出链全称。
进入 /blockchain-config

币种扩展参数

Core
维护链参数：RPC、归集地址、提币地址、精度与扩展字段 JSON。
进入 /coin-chain-config
推荐阅读：README.md、README-coin-config.md。项目已收敛为纯配置中心能力。
